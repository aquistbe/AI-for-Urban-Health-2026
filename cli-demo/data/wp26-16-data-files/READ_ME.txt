Tract-to-Districts Crosswalk

This crosswalk accompanies the working paper "Off the Beaten Tract: Constructing a New Neighborhood Geography Using Revealed Preference," available at https://www.philadelphiafed.org/community-development/housing-and-neighborhoods/off-the-beaten-tract-constructing-a-new-neighborhood-geography-using-revealed-preference.

Recommended citation: Mast, Evan and Alaina Barca. 2026. "Off the Beaten Tract: Constructing a New Neighborhood Geography Using Revealed Preference." Federal Reserve Bank of Philadelphia Working Paper No. 26-16.


Variables

- geoid10: 2010 tract GEOID FIPS code.
- district_id: ID of tract's district.
- place_fips: Tract's 2010 place FIPS code.
- countya: Tract's 2010 county FIPS code.
- statea: Tract's state FIPS code.
- cbsa10, cbsa_title: Tract's 2010 CBSA FIPS code and title, respectively.
- low_pop_flag: Flag if tract population is less than 500.
- isolation_flag: Flag if tract does not border other tracts (e.g., islands).
